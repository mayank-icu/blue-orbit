## Overview
Blue Orbit is an interactive web application built with React. It features an AI-powered chat assistant and engaging features that help users explore space and the broader cosmic horizons. 

## Getting Started

Running this project locally is super simple. Just follow these steps:

1. **Install Dependencies**  
   Open your terminal in the project folder and run:
   ```bash
   npm install
   ```

2. **Set Up Your Environment**  
   Make sure you have a `.env` file in the root of the project folder. You'll need to add your Gemini API key like this:
   ```env
   REACT_APP_GEMINI_API_KEY=your_actual_api_key_here
   ```

3. **Start the App**  
   Run the following command to spin up the development server:
   ```bash
   npm start
   ```
   
   The app should automatically pop open in your default browser at `http://localhost:3000`.

Enjoy exploring!
